부모 요소에display:grid 적용시 자식 요소들은 grid cell로 변함

개념

grid(부모요소)

  • 가로 사이즈 설정: grid-template-columns

  • 세로 사이즈 설정: grid-template-rows

  • 레이아웃 설정: grid-template-areas

  • 간격 설정-전체: grid-gap

  • 간격 설정-별도:grid-column-gap / grid-row-gap

grid cell(자식요소)

  • 특정 item을 표시하기 시작할/끝마칠 열 지정: grid-column-start / grid-column-end

  • 특정 item을 표시하기 시작할/끝마칠 행 지정: grid-row-start / grid-row-end

  • 레이아웃 설정: grid-template-areas

  • 간격 설정-전체: grid-gap

  • 간격 설정-별도:grid-column-gap / grid-row-gap
